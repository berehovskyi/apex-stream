# v3.3.1

## 🚀 New Features:

- Added new String Predicates:
    - `iLike` - case-insensitive LIKE pattern matching

## 🔄 Changes
- Renamed String Predicates for better clarity and consistency:
  
  | Old Method Name  | New Method Name  | Description |
  |------------------|------------------|-------------|
  | `containingIc`   | `iContaining`    | Checks if string contains the specified substring (case-insensitive) |
  | `startingWithIc` | `iStartingWith`  | Checks if string starts with the specified prefix (case-insensitive) |
  | `endingWithIc`   | `iEndingWith`    | Checks if string ends with the specified suffix (case-insensitive) |
  |                  | `iLike`          | New method for case-insensitive LIKE pattern matching |

📢Note: The existing `likex` method now performs case-sensitive matching by default.
